1. Changing a background and opacity

These buttons call a JavaScript function from main.js to change the background and opacity of the box.

JavaScript

1. An object is created with document.getElementById()

2. JS adds a object.style.transition to object

3. JS adds a object.style.background or style.opacity property

JavaScript

3. Animations with KeyFrames

Animation

1. The CSS has a box animating between green and yellow when the page loads.

2. The "Animate box" button does not create a new variable, but simply uses document.getElementById("the Id").style.animation="params" in order to create a new animation sequence for this box.

3. The animation sequences are stored in main.css as @keyframes
