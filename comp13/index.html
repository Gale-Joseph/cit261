Transformations

This page demonstrates the basics of employing the transform property in CSS.

Summary: There are two basic steps for writing an animation:
a. write your animation properties on the element to animate and include the name of the corresponding keyframe
b. write your @keyframe process and name it.
The animation syntax goes in the element and is typically applied immediately like transform. If using psuedo elements, put animation statements under pseduo class.
