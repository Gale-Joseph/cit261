Competency 10: Audio Tags, Video Tags, Canvas Tag

Audio Tags

These three audio tags practice using the attributes of autoplay,loop, and muted.

Audio tag, autoplay attribute

This audio tag is set to autoplay.

Audio tag, loop attribute

This audio tag is set to loop and will play until stopped.

Audio tag, muted attribute

This audio tag is set muted, meaning it will not play until unmuted.

Video Tags

These two video tags practice using the attributes of loop and muted.

Video tag, loop attribute

This video tag is set to loop.

Video tag, muted attribute

This video tag is set to muted.
