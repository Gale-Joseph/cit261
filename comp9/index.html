Including those for Mobile Devices ( Ex. onTouchBegin, onLoad, etc.)
and Animation and Transition Events.

The following are a sample of standard JavaScript events:

onafterprint

This event will print message after a print dialogue box is closed. Go ahead and hit Control + P and close the box to see what happens.

onreset

When a form is reset, a dialogue box will appear.

onkeypress

When a key is pressed in an input box a popup will appear
